Git/GitHub

git init

git add .

git commit -m "Initial commit"

git branch -M main

git remote add origin https://github.com/USERNAME/REPO_NAME.git

git remote add origin https://github.com/RSJK2562/Git-GitHub.git

git push -u origin main

अपनी लोकल ब्रांच का नाम "main" में बदलें और फिर पुश करें:
1. git branch -m master main
2. git push -u origin main

अगर आपने GitHub पर README, लाइसेंस, या .gitignore के साथ एक खाली रिपॉजिटरी बनाई है, तो आपको पहले उन परिवर्तनों को पुल करना होगा:
git pull origin main --allow-unrelated-histories

Check Remote Repositories: (git remote -v)

Remove the Remote: git remote remove origin
