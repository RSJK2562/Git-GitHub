Git Branching Example

This page demonstrates how to work with branches in Git.

Creating a New Branch

To create a new branch, use the following command:

git branch new_branch_name

Switching Branches

To switch to the newly created branch, use:

git checkout new_branch_name

Merging Branches

To merge changes from one branch into another, first switch to the target branch and then use:

git merge branch_name

Pushing Changes

After making changes, you can push your branch to the remote repository:

git push -u origin new_branch_name
