Multi-Branch Work

To work with multiple branches in Git, you can follow these steps:

  1. Create a new branch: git branch branch_name
  2. Switch to the new branch: git checkout branch_name
  3. Make changes and commit them:
    • git add .
    • git commit -m "Your commit message"
  4. Push the new branch to the remote repository: git push -u origin branch_name

To switch back to the main branch, use: git checkout main

To merge changes from one branch into another, first switch to the target branch and then use: git merge branch_name

To delete a branch after merging, use: git branch -d branch_name
